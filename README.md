# Expensify - Expense Tracker

A simple and responsive Expense Tracker web application built using **HTML, CSS, and JavaScript**. This project helps users track their income and expenses,
calculate their current balance, and maintain a transaction history in an easy-to-use interface.

## Features

- Add income and expense transactions
- Automatic balance calculation
- Separate income and expense totals
- Transaction history display
- Responsive and modern UI design
- Real-time updates using JavaScript
- Simple and lightweight implementation

## Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and responsive design
- **JavaScript (ES6)** – Application logic and DOM manipulation

## How It Works

1. Enter the transaction name.
2. Enter the transaction amount:
   - Positive value (`+`) for Income
   - Negative value (`-`) for Expense
3. Click **Add Transaction**.
4. The application will:
   - Update the total balance
   - Update income and expense totals
   - Add the transaction to the history list
